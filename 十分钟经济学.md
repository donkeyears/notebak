# 垄断

横向垄断，占有市场份额大。

纵向垄断，产业链控制能力大

专利是一种偏好的垄断。但有有效期。

自然垄断：类似电力，一个国家三种电肯定不好用，这时候政府可以通过控制价格来管控。

价格歧视：不同客户不同价格。比如双十一

担心垄断的根源是担心价格制定的垄断。



# 博弈论

两个对手可以定价格。如果都高都可以挣钱，但是对于一个人来说，他定低了就可以多挣钱，他会降价。另一个人也这么想。最后俩人价格都低了。
